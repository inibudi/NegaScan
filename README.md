# NegaScan
Negatif content website detection system is based on a machine learning method.

Requirements:
1. Python 3.13
2. Visual Studio Code as IDE
 
Libraries:
1. joblib:
   `pip install joblib`
2. requests:
   `pip install requests`
3. bs4:
   `pip install bs4`
4. scikit-learn:
   `pip install scikit-learn`
5. nltk:
   `pip install nltk`
6. pandas:
   `pip install pandas`
